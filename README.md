# Base Portfolio Tracker

A Web3 dashboard built on Base network to track wallet balances, tokens, and transaction history.

## Features

- Wallet connection (wagmi)
- ETH balance display
- Token tracking (Alchemy API)
- Transaction history
- BaseScan integration

## Tech Stack

- React + Vite
- Tailwind CSS
- Wagmi + Viem
- Alchemy API

## Setup

```bash
npm install
npm run dev
```
